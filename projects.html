TaskFlo - Task Management App

TaskFlo is a web app designed to help users organize their tasks and folders efficiently. It features real-time data handling with Firebase.

  • Secure login with Firebase Authentication
  • Create, edit, and delete tasks within folders
  • Real-time sync via Firebase Firestore
  • Offline support with local storage
  • Deadline tracking & status updates
Tech Stack: HTML, CSS, JavaScript, Firebase (Firestore, Auth, Hosting), Node.js, VS Code, Git/GitHub.
